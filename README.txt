# Jogo da Cobrinha
### Membro: Oscar Alves Jonson Neto
### Projeto: TP Programação Modular

Uma implementação clássica do jogo da cobrinha em Python, onde você controla uma cobra que cresce ao comer comida, enquanto evita colidir com as paredes e com seu próprio corpo.

# Dependências

Python 3.x
Biblioteca Pygame

# Instalação

Certifique-se de ter o Python 3.x instalado.
Para instalar o Pygame, você pode usar o pip. Abra o terminal ou prompt de comando e execute o seguinte comando:

Windows: py -m pip install -U pygame --user

Debian: sudo apt-get install python3-pygame
